Configuration

  • General

    • Statistics

      • Show statistics after analysis

        When finished, open a popup window with the results of the analysis. The popup is shown only when there is one of these categories detected: broken, invalid or timeout.

      • Automatically hide statistics

        Hide statistics after a defined amount of seconds. Use 0 to disable this setting.

    • Highlight color for analyzed links

      • Fine links

        Color to highlight links that work.

      • Broken links

        Color to highlight broken links.

      • Time out links

        Color to highlight links that didn't return any headers for a long time

      • Skipped links

        Color for highlighting links weren't analyzed

      • Invalid links

        Highlight color for links that returned header, but it wasn't http/https header

  • Advanced

    • Time-out (in seconds)

      Links that take longer than the defined threshold are being considered as timed out

    • HTML tags to be analyzed

      • Anchor

      • Option

      • Area


    • Skip links containing

      Separate by ','
